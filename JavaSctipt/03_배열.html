배열

1. 배열 선언 및 초기화

    자바스크립트에서 배열은 자료형을 지정하지 않기때문에 한 배열 안에 모든 자료형의 데이터 보관이 가능하다.
    배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 크기를 정하고 선언할 수도 있다.

2. 배열의 메소드

    indexOf()
    메소드 개요: 배열에서 특정 요소의 인덱스를 찾습니다.
    각 파라미터의 역할: 찾을 요소를 인수로 전달합니다.
    리턴타입: 찾은 요소의 인덱스를 반환하며, 요소가 없을 경우 -1을 반환합니다.
    원본배열의 변화: 원본 배열에 변화를 주지 않습니다.

    concat()
    메소드 개요: 하나 이상의 배열을 기존 배열에 연결하여 새로운 배열을 반환합니다.
    각 파라미터의 역할: 연결할 배열을 인수로 전달합니다.
    리턴타입: 연결된 새로운 배열을 반환합니다.
    원본배열의 변화: 원본 배열에 변화를 주지 않습니다.

    join()
    메소드 개요: 배열의 모든 요소를 하나의 문자열로 합쳐서 반환합니다.
    각 파라미터의 역할: 구분자 문자열을 인수로 전달할 수 있습니다. 기본값은 쉼표(,)입니다.
    리턴타입: 합쳐진 문자열을 반환합니다.
    원본배열의 변화: 원본 배열에 변화를 주지 않습니다.

    reverse()
    메소드 개요: 배열의 요소 순서를 반전시킵니다.
    리턴타입: 반전된 배열을 반환합니다.
    원본배열의 변화: 원본 배열이 반전됩니다.

    sort()
    메소드 개요: 배열의 요소를 정렬합니다. 기본적으로는 사전순으로 정렬하며, 변경된 배열을 반환합니다.
    리턴타입: 정렬된 배열을 반환합니다.
    원본배열의 변화: 원본 배열이 정렬됩니다.

    push()
    메소드 개요: 배열의 끝에 하나 이상의 요소를 추가합니다.
    각 파라미터의 역할: 추가할 요소를 인수로 전달합니다.
    리턴타입: 배열의 새로운 길이를 반환합니다.
    원본배열의 변화: 원본 배열이 변경됩니다.

    pop()
    메소드 개요: 배열의 마지막 요소를 제거하고 해당 요소를 반환합니다.
    리턴타입: 제거된 마지막 요소를 반환합니다.
    원본배열의 변화: 원본 배열이 변경됩니다.

    shift()
    메소드 개요: 배열의 첫 번째 요소를 제거하고 해당 요소를 반환합니다.
    리턴타입: 제거된 첫 번째 요소를 반환합니다.
    원본배열의 변화: 원본 배열이 변경됩니다.

    unshift()
    메소드 개요: 배열의 맨 앞에 하나 이상의 요소를 추가합니다.
    각 파라미터의 역할: 추가할 요소를 인수로 전달합니다.
    리턴타입: 배열의 새로운 길이를 반환합니다.
    원본배열의 변화: 원본 배열이 변경됩니다.

    slice()
    메소드 개요: 배열의 일부분을 선택하여 새로운 배열을 반환합니다.
    각 파라미터의 역할: 시작 인덱스와 끝 인덱스를 인수로 받습니다. 끝 인덱스는 선택된 요소에 포함되지 않습니다.
    리턴타입: 선택된 요소로 이루어진 새로운 배열을 반환합니다.
    원본배열의 변화: 원본 배열에 영향을 주지 않습니다.

    splice()
    메소드 개요: 배열의 요소를 추가하거나 제거하고, 변경된 요소들을 반환합니다.
    각 파라미터의 역할: 시작 인덱스, 제거할 요소의 개수, 추가할 요소들을 인수로 받습니다.
    리턴타입: 제거된 요소들로 이루어진 배열을 반환합니다.
    원본배열의 변화: 원본 배열이 변경됩니다.
